Projects

The Classroom Voter - PYTHON VERSION 1.1

The Classroom Voter is a python based app which helps to elect your leaders. The Voting App first opens with a Login and Registration window where you can choose yur user type. There are also two dashboards depending on whether you are a Student or Teacher. On the Home Dashboard you can view features such as the voterooms, profile and results. The Teacher can access exclusive features such as creating voterooms and checking the live results for the vote rooms as well as graphs. The Student can have excess features such as voting on the rooms as well as access to graphs. When you are a teacher you can create a unique code which is important as it allows students to vote as well as gives you access to the results and graphs. Then, you can create a vote room with the topic and number of candidates then writing the names and details of each candidate. As a student you can vote anonymously with the details present on the window to help you decide. The teacher can then access the results live as well as a pie chart which is accessible to both teachers and students. Please be noted that the application has not yet been turned into a public application so this will only work for a single device and store databases locally. View Project
